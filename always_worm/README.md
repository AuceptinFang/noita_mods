# Always Worm 模组

## 功能
让混沌变形药水只会将你变成各种蠕虫，而不是随机生物。

## 蠕虫类型
混沌变形现在会随机选择以下五种蠕虫之一：
- **普通蠕虫** - 标准的蠕虫
- **大蠕虫** -
- **小蠕虫** - 
- **地狱蠕虫** - 
- **幽灵蠕虫** - 

## 安装
1. 将此模组文件夹放置在 `mods/` 目录下
2. 在游戏中启用模组，名称为 `Always Worm`
3. 重启游戏

## 使用
启用模组后，所有混沌变形效果（如混沌变形药水）都只会将你变成上述五种蠕虫中的一种。

## 原理
模组通过修改游戏的polymorph表，将所有随机变形目标替换为五种蠕虫实体。

## 兼容性
- 会覆盖其他修改polymorph表的模组
